# concat-mp4

> An electron-vue project

批量无损合并MP4两个文件，两个文件必须有相同的参数。
主要用于批量加片头。

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
